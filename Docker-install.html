DOCKER INSTALLATION MATRIX

Linux (Ubuntu/Debian)

Step 1: Update the System

Open your terminal and run:

sudo apt update

Step 2: Install Required Packages

sudo apt install apt-transport-https ca-certificates curl software-properties-common

Step 3: Add Docker's GPG Key

curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo tee /usr/share/keyrings/docker-archive-keyring.gpg > /dev/null

Step 4: Add Docker's Repository

echo "deb [arch=amd64 signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

Step 5: Install Docker

sudo apt update
sudo apt install docker-ce

Step 6: Verify Installation

sudo systemctl status docker

Windows

Step 1: Download Docker Desktop

Visit the official Docker website and download Docker Desktop for Windows

Step 2: Install Docker Desktop

  • Run the installer you just downloaded
  • Follow the on-screen instructions
  • Make sure "Use WSL 2 based engine" is selected (if prompted)
  • Accept default settings unless specific customization is needed

Step 3: Launch Docker Desktop

Find Docker Desktop in your Start menu and launch it

Step 4: Complete Setup

Wait for the initial setup to complete. Look for the Docker whale icon in your taskbar.

macOS

Step 1: Download Docker Desktop

Visit the official Docker website and download Docker Desktop for Mac

Step 2: Install Docker Desktop

  • Open the downloaded .dmg file
  • Drag the Docker icon to Applications folder
  • Launch Docker from Applications

Step 3: Complete Installation

Docker will automatically finish setup the first time you open it. Look for the Docker whale icon in your menu bar.

Step 4: Verify Installation

docker --version

Post-Installation Verification (All Systems)

Test your Docker installation by running:

docker run hello-world

This command downloads a test image and runs it in a container. If successful, you'll see a message confirming that Docker is working correctly.
